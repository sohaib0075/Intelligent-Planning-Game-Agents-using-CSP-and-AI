# Intelligent Planning Game Agents using CSP and AI

An AI project that implements **intelligent game-playing agents** using **Constraint Satisfaction Problems (CSP)** and AI planning techniques to solve and navigate game environments autonomously.

---

## Overview

This project explores how AI agents can be designed to make intelligent decisions in game environments using CSP-based planning. Constraint Satisfaction is used to model game rules and goals, allowing agents to plan optimal moves and strategies.

## Key Concepts

- **CSP (Constraint Satisfaction Problem)** — modeling game rules as constraints
- **AI Planning** — agents that reason about future states to make decisions
- **Game Agents** — autonomous agents that play/solve games intelligently
- **Backtracking Search** — exploring possible moves within constraints
- **Heuristics** — optimizing agent performance with smart pruning

---

## Features

- Intelligent agent design using CSP formulation
- Constraint-based move validation
- Planning and decision-making in game environments
- Visualization of agent behavior and game states
